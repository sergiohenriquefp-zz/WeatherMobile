# WeatherMobile
> Basic integration with Open Weather API to try some UnitTest and UITesting

## Before Running
>Go to the root of the project
>See the [cocoapods](https://cocoapods.org).
`$ pod install`

## Running
>Inside Xcode type
`$ command+R`

## Testing
>Inside Xcode type
`$ command+U`
